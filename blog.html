What are the different ways to select an element in the DOM?
We can select an element in different ways such as getElementById, getElementsByClassName, querySelector, querySelectorAll and so on.
What is the difference between innerHTML, innerText, and textContent ?
innerHTML gets the HTML element inside the HTML tag and sets the inside HTML element. innerText gets the inside text of HTML element and sets the inside visible text. textContent gets the all text content including hidden text and sets the all text content.
What is event delegation in the DOM?
Event Delegation is a technique where we can add a event listener to a parent element. It is a power technique to improves efficiency when dealing with many child elements.
What is event bubbling in the DOM?
Event bubbling is a process where an event starts from the target element and propagates up through its parent elements until it reaches the root.
How do you create, add, and remove elements using JavaScript?
We use document.createElement() to create a new element and use appendChild() to add a newly created element. We use .remove() to remove a elementfrom the DOM.
